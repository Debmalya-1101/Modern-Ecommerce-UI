# Angular Project Instructions

## Project Goal

Personal Angular project with enterprise-relevant architecture and modern e-commerce UI.

## Tech Stack

- Latest Angular
- Angular Material
- Standalone Components
- SCSS
- RxJS
- Angular Signals where appropriate

## Architecture Rules

- Preserve existing architecture
- Do not rewrite unrelated files
- Prefer maintainability over clever code
- Use strict typing
- Use meaningful naming
- Use feature-based structure

## UI Rules

- Modern e-commerce feel
- Responsive design
- Angular Material
- Professional spacing and typography
- Mobile-friendly

## Implementation Rules

Every feature must include:

- UI
- Routing
- Navigation access (if applicable)
- Loading state
- Error state
- Responsive design

A feature is not complete unless it can be tested from the browser.

## Learning Rules

After implementation:

- Update LEARNING_JOURNAL.md
- Explain concepts simply
- Assume backend-heavy developer
- Include small examples
